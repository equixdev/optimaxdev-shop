# Optimax Dev Shop

![Screenshot](screenshot.png)

A store cart emulator application built with:
- React
- Redux Toolkit
- TypeScript
- create-react-app

Features:
- Product browsing and filtering
- Shopping cart with Redux state management
- Reusable UI components (Button, Input, NumberInput, Cart)

## Getting Started

### Prerequisites
- Node.js v16+ (LTS versions recommended)
- npm or yarn

### Installation & Running

```bash
npm i
npm start
```

The app opens at `http://localhost:3000` with hot reload on code changes.

### Building for production

```bash
npm run build
```

The production build outputs to `build/` and is ready to deploy.
